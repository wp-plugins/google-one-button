Provides a +1 button from Google+ beneath each post/page.

== Description ==

This plugins enables the famous Google +1-button in your wordpress-installation. The button is shown beneath
your posts/pages. With this button you can share the permalink to your post/page in your Google+ profile.

You've got the possibility to change the look of the button in the same manner, that you can do by using the
configurator that Google provides: see [Google plus one](http://www.google.com/webmasters/+1/button/).

== Installation ==

1. Upload `googleplusone.php` to the `/wp-contents/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in Wordpress
3. Configure the look & feel of the button in the settings-section of the backend

== Frequently Asked Questions ==

= Does this plugin need any additional database-tables? =

No this plugin doesn't need any additional database-tables. The configuration of the plugin however is stored within
the `options`-table
